## About the Project

This is my personal portfolio website built as the course project for the LUT SDS Front-End module. The site showcases my skills, projects, and a bit about me as a CS student.

The website includes:
- Responsive navigation with a mobile hamburger menu
- Hero section, About, Projects, Skills, Slideshow, FAQ accordion, and Contact form
- CSS Grid and Flexbox layouts
- Pure JavaScript for interactivity (no frameworks)
- Responsive design with media queries
- Deployed via GitHub Pages

---

## How to Run the Project

### Option 1 – Open directly (simplest)
1. Download or clone this repository
2. Open `index.html` in any web browser
3. That's it – no build step or server needed

### Option 2 – Use VS Code Live Server
1. Install [VS Code](https://code.visualstudio.com/)
2. Install the **Live Server** extension
3. Open the project folder in VS Code
4. Right-click `index.html` → **Open with Live Server**
5. The site opens at `http://127.0.0.1:5500`

### Option 3 – View the live deployed site
The site is deployed at GitHub Pages:  
👉 **https://sandippgautam.github.io/Software-Development-Skills-Front-End-2025-26-Sandip_Gautam/**
